# Paper Landscape — DEPRECATED

> **Status: DEPRECATED 2026-05-04.** This document was retired during the post-fan-out integration pass that
> produced the 13 thematic syntheses in [`docs/syntheses/`](../syntheses/) and the 10 cluster syntheses in
> [`docs/syntheses/repo-clusters/`](../syntheses/repo-clusters/). Its narrative content has been absorbed into
> those syntheses; its navigation function moved to [`docs/papers/INDEX.md`](../papers/INDEX.md).

## Where the content went

- **Per-paper notes:** [`docs/paper-notes/`](../paper-notes/) (78 files, unchanged).
- **Navigation lookup** (paper → covering syntheses): [`docs/papers/INDEX.md`](../papers/INDEX.md).
- **Thematic groupings** (BitNet thread, larger-than-RAM streaming, KB substrate, memory pillar, etc.): the
  thematic syntheses in [`docs/syntheses/`](../syntheses/), each focused on one theme rather than the survey-style
  overview this document used to provide.
- **Cross-cutting picture:** [`docs/landscapes/total-landscape.md`](total-landscape.md) and
  [`docs/landscapes/synthesis-landscape.md`](synthesis-landscape.md).
- **Reading orders by Linus phase:** absorbed into the relevant thematic syntheses; the "Where this synthesis
  fits" section at the end of each names its phase relevance.

## Why it was retired

When this document was written, papers were the primary unit of analysis and the syntheses were an optional layer
on top. After the post-fan-out work, the syntheses became the primary unit and the per-paper structure was
duplicating their content with stale framings. The current architecture treats syntheses as the load-bearing layer
and uses indexes for navigation; this document had no remaining role.

This stub preserves link integrity for any document that still references `paper-landscape.md`.
